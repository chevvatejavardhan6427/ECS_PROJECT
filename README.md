Overview

LifeLink is my first ECS (Engineering Computing Skills) project. It is a web-based platform designed to connect blood donors with people in need of blood. The goal of this project is to make the process of finding blood donors faster and more efficient while also promoting awareness about organ donation.

The platform allows users to register as donors, search for nearby donors using location services, view blood donation camps, and learn more about the importance of blood and organ donation.

Problem Statement

During emergencies, finding a blood donor can be a difficult and time-consuming process. People often rely on personal contacts or social media to search for donors, which may delay treatment.

LifeLink aims to address this problem by creating a centralized platform where donors and recipients can connect quickly and efficiently.

Development Methodology

This project was developed using the Agile methodology.

Instead of building the entire application at once, the project was developed in small stages. Features were added step by step, tested, and improved throughout the development process. This approach allowed flexibility and continuous improvement while building the platform.
